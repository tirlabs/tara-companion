# DESIGN_BRIEF_V1.md

## Objective

Design the MVP mobile experience for Tara.

## Inputs

* TARA_VISION.md
* TARA_MVP.md
* TARA_USER_JOURNEY.md
* TARA_DESIGN_SYSTEM.md

## Deliverables

* Home Screen
* Listening Screen
* Thinking Screen
* Speaking Screen
* Conversation Screen

## Design Goals

* Companion-first
* Conversation-first
* Mobile-first
* Friendly
* Calm
* Trustworthy

## Design Constraints

* No LMS appearance
* No dashboard appearance
* No analytics
* No assessment-focused UI
* No ChatGPT clone appearance

## Required Concepts

* Concept A: Companion First
* Concept B: Voice First
* Concept C: Future Learning Companion

## Success Criteria

A learner should immediately feel:

"I have someone I can talk to."

rather than:

"I am using educational software."
